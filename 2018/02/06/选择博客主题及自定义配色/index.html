选择博客主题及自定义配色

 2018/02/06   Share

博客主题

Hexo 的默认主题比较简陋，但是好在网上可以找到很多其他主题，本博客在 Hexo 官网选择了这款 archer 主题。主题的安装方式很简单，通常作者都有具体的讲解。

archer 的默认配色

直观地看， archer 主题的默认配色以灰色为主，粉色为辅(抱歉我对色彩没什么研究，不知道是哪种粉色)。我简单拉了一下 archer 默认配色表，如下图所示：

可以看出作者采用的色搭一明一暗，虽然简约但色阶非常丰富。其中，由于我是用手工检查配色的——主要是通过微信截图功能检查各区域的颜色，以及检查 \archer\source-src\scss 文件夹下的各配置文件，因此有所疏漏在所难免。我个人对这套配色的理解是：

  • 由于每个人对颜色的偏好各有不同，有的人喜欢蓝色，有的人喜欢红色，主题的作者为了适应大多数人的口味而将主色调设为灰色（不偏倚任何颜色）。
  • 灰色为底色便于阅读，而且丰富的色阶使灰色不显单调。
  • 作者给出的自定义配色方法只针对粉色，粉色主要出现在一些标签上、鼠标悬停时以及选中文本的背景色（按 ctrl+A ）。简单地改变这一种颜色，就可以改变除灰色外的主体颜色，体现不同使用者的色彩倾向。

我的配色计划

就我个人而言，偏好中国古代建筑（尤其是宗教建筑）中的一些色彩，比如枣红、藏蓝、绿釉、金漆等等。因此，我参考了这些颜色，并小幅地拉开色阶，如下图所示。

当中我的考虑是：

  • 蓝色主要用于文本，红色用于背景，黄色用于高亮，绿色候补。
  • 每种颜色只简单拉开少数色阶，便于我这个外行取舍（我驾驭不了太多颜色）。
  • 每种颜色的主力色（标记为主要的四个颜色）都选择纯度较低的颜色，视觉上比较温和。

我的实际配色

由于没有任何前端知识，寻找这些对应区域的对应颜色颇费了一番功夫，有种刀耕火种的感觉，果然没有知识就没有生产力啊。其中我咨询过一次 archer 主题的作者，也略微参考了一些教程，比如：

  • 从零开始制作 Hexo 主题
  • 写一个自己的Hexo主题
  • Hexo高级教程之主题开发

最终，我采取的配色方案整体效果就如本博客目前所展示的这样，其中备胎绿并没有得到使用，此外也沿用了默认配色的一些灰色。具体各部分可作如下展示：



后记

几天下来，颜色调整了很多次，最后也更换了几张头图，博客就焕然一新了。一点点体会就是，网页的颜色不宜过多，比如豆瓣主绿、知乎主蓝、政府网站主红等等，这也就是为什么我实施时没有用到绿色。
最后，附上一张图简要对比 archer 默认配色和我的配色，诸君可自行体会。

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: false
    tags: true
